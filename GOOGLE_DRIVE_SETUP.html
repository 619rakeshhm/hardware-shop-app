Saving Your Data to Google Drive

Automatic cloud backup that still works offline. The app saves instantly to a file on your PC, and Google Drive syncs that file to the cloud in the background.

Works in: Google Chrome, Microsoft Edge, or Brave. (Firefox and Safari can’t auto-save to a drive — use Download Backup there.)

One-time setup (about 5 minutes)

1Install Google Drive for Desktop

  1. Go to https://www.google.com/drive/download/
  2. Download and install Google Drive for Desktop.
  3. Sign in with the Google account you want your shop data saved to.
  4. You’ll see a new drive in File Explorer, usually Google Drive (G:) with a My Drive folder.
Tip: No G: drive? Open the Drive icon in the system tray (bottom-right) → gear → Preferences → enable file streaming.

2Make a folder for the shop data (optional)

  1. Open File Explorer → Google Drive (G:) → My Drive.
  2. Create a folder named Dukaan Desk.

3Point the app at a file in Google Drive

  1. Open the app (index.html) in Chrome or Edge.
  2. Go to Settings → Data.
  3. Click Save To Drive.
  4. Navigate to Google Drive (G:) → My Drive → Dukaan Desk.
  5. Keep the suggested name and click Save.
  6. The status should read “Selected: …json. Auto-save is on.” and the top pill should turn green: “Drive auto-save on.”

✅ That’s it — every bill, stock change, and ledger entry now saves automatically and syncs to the cloud.

Daily use

  • Just use the app normally — you don’t need to click anything to save.
  • The green “Drive auto-save on” pill confirms it’s connected.
  • If it ever shows “Drive auto-save off” or “Auto-save paused”, click Set Auto-Save (or Save Again) and re-select your file. This can happen after the browser’s been closed a while — it’s normal.

Opening your data on another PC

  1. Install Google Drive for Desktop on the second PC, sign in with the same account, and wait for it to sync.
  2. Copy the app folder (with index.html) to that PC and open it.
  3. Go to Settings → Data → Open Data File and pick your …data….json from the Google Drive folder.
  4. Click Set Auto-Save so changes on this PC sync back too.
Use one PC at a time. If two PCs edit the same file at once, Google may create a “conflicted copy.” For a single shop counter this is rarely an issue — just don’t edit on two machines simultaneously.

Extra safety (recommended)

Once a week, click Settings → Data → Download Backup and keep that .json on a USB stick or second folder. It’s a manual snapshot you can Restore Backup from if anything ever goes wrong with the synced file.

Troubleshooting

ProblemFix
No G: / Google Drive folder appearsDrive tray icon → Preferences → enable file streaming; restart the PC
Pill says “Drive auto-save off”Settings → Data → Set Auto-Save, pick your file again
“Auto-save paused. Click Set Auto-Save to reconnect.”Click Set Auto-Save and allow file access when asked
File picker won’t open / button does nothingYou’re likely in Firefox/Safari — use Chrome or Edge, or use Download Backup
Data didn’t appear on the second PCWait for Google Drive to finish syncing, then Open Data File
Confirm it’s syncingIn File Explorer the file shows a green check (synced) rather than a sync arrow

How it works

The app saves to a normal file on your PC using your browser’s secure file access. That file lives inside the Google Drive folder, so Google Drive for Desktop syncs every change up to the cloud automatically — giving you cloud backup while the app stays fully offline-capable.
